Samuel Morse, in collaboration with Alfred Vail, introduced **Morse code** in **1838**, revolutionizing long-distance communication by enabling messages to be transmitted rapidly using electrical signals.

The system represents characters using **dots (dits)** and **dashes (dahs)** governed by strict timing rules:

- **Dit**: 1 time unit  
- **Dah**: 3 time units  
- **Pause between letters**: 3 time units  
- **Pause between words**: 7 time units  

These timing conventions are fundamental to accurate encoding and decoding.

## Key Features

### Letters and Signals

Each letter is encoded as a unique sequence of dots and dashes. For example:

- **S** → `...`  
- **O** → `---`

The most well-known signal is **SOS** (`... --- ...`), adopted internationally as a distress signal due to its simplicity and unmistakable rhythm.

### Efficiency in Design

Morse code was intentionally optimized for efficiency. Frequently used letters, such as **E**, were assigned the shortest possible representations. This design minimized transmission time and improved overall communication speed.

### Timing Precision

Accurate timing is essential in Morse code. Even small deviations in spacing or signal length can alter meaning, making disciplined rhythm a critical skill for operators.

### Speed Calculation

Transmission speed is typically measured in **words per minute (WPM)**. The standard reference word, *“Paris”*, is defined as **50 time units**. At **20 WPM**, this corresponds to approximately **2.5 seconds per word**, highlighting the importance of precise timing for consistent communication rates.

## Historical Context and Impact

Morse code overcame the limitations of earlier visual and mechanical signaling systems, enabling reliable communication across vast distances. Its international adoption made it a cornerstone of early telecommunications, particularly in maritime and emergency contexts.

Although largely supplanted by modern digital technologies, Morse code remains a foundational milestone in communication history and continues to be used in specialized applications and amateur radio.
